INSTRUCTIONS FOR USE:
> Upon loading, select an object to render by clicking 'choose a file'.
> Select one of the .obj files from 3D Engine 3.1\3D Engine 3.1\bin\Debug\Meshes
> Use WASD, C, Space, and the mouse to move the camera.
> Press ESC at any time to show or hide the mouse.
> You may choose to load another file by selecting that option from the top left.
> You may change the position and set a rotation for the object by selecting 'Transformations' at the top left.
> Recommended: Select icosphere.obj and give it a rotation vector for a cool effect.
